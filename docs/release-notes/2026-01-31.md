# Release Notes - 2026-01-31

## Added
- **Multi-tool navigation**: New tabbed navigation in the header to switch between tools.
- **Status Code Validator**: New tool to batch-check HTTP status codes for multiple URLs.
  - Paste URLs (one per line, comma, or space separated).
  - Summary dashboard showing 2xx, 3xx, 4xx, 5xx, and error counts.
  - Filter results by URL text or status category.
  - Export results to CSV.
- Railway deployment configs and Chromium settings for Puppeteer.
