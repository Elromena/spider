# Release Notes - 2026-01-31

## Added
- **Multi-tool navigation**: New tabbed navigation in the header to switch between tools.
- **Status Code Validator**: New tool to batch-check HTTP status codes for multiple URLs.
  - Paste URLs (one per line, comma, or space separated).
  - Summary dashboard showing 2xx, 3xx, 4xx, 5xx, and error counts.
  - Filter results by URL text or status category.
  - Export results to CSV.
  - **Save reports** with shareable links for team collaboration.
  - **Track progress** with "done" checkboxes per item.
  - **Re-check on mark done** to verify and update the status code.
- Railway deployment configs and Chromium settings for Puppeteer.

## Fixed
- Remove temporary debug instrumentation that could recreate `.cursor/debug.log`.

## Improved
- **Full responsive design** across the entire application:
  - Main app layout adapts to all screen sizes (mobile, tablet, desktop).
  - Link Auditor: Sites panel becomes horizontal scroll on mobile, forms stack vertically.
  - Status Validator: Two-panel layout collapses to single column on mobile.
  - Health Report modal popup: Stats wrap to 2 columns, buttons stack vertically.
  - report.html (full health report page): Filter buttons stack, stat cards wrap.
  - status-report.html: Summary cards and filter inputs adapt to smaller screens.
  - Touch-friendly improvements: Larger tap targets, 16px font on inputs to prevent iOS zoom.
  - Tool navigation becomes fixed bottom bar on mobile.
  - Landscape mode optimizations for mobile devices.
  - All modals and dropdowns properly sized for mobile.
